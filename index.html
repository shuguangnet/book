简单为先

通过以 Markdown 为中心的项目结构实现最小化设置，帮助你专注于写作。

基于 Vue 的强大支持

享受 Vue 的开发体验，可以在 Markdown 中使用 Vue 组件，并用 Vue 开发自定义主题。

高性能

VuePress 为每个页面生成预渲染的静态 HTML，并在页面加载后作为单页应用程序 (SPA) 运行。

多样化的主题

提供了开箱即用的默认主题。你也可以选择一个社区主题或创建自己的主题。

灵活的插件

提供灵活的插件 API，允许插件为你的网站提供丰富的即插即用功能。

支持多种打包工具

默认打包工具是 Vite，同时也支持 Webpack。选择你喜欢的工具！
